How RAG Works

1
Upload a document — text is extracted and split into chunks
2
Each chunk is converted to a vector (embedding) using HuggingFace
3
Chunks are stored in a FAISS vector store (in-memory)
4
Your question is embedded and matched via similarity search
5
Top matching chunks + your question are sent to Groq LLM
6
Answer is generated only from the retrieved document context
